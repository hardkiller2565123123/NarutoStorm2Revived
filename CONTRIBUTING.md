# Contributing

Thank you for wanting to contribute.

## How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes.
5. Open a Pull Request.

## Rules

- Keep changes clean and organized.
- Explain what your change does.
- Do not submit stolen or leaked code.
- Do not include copyrighted game files.
- Respect the purpose of the project.

## Good Contributions

- Bug fixes
- Documentation
- Modding research
- Tool improvements
- Compatibility fixes
- UI improvements
